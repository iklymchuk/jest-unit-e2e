## 	Jest unit and e2e test automation sample

**Installation**

``
npm init
``

``
npm install --save-dev jest
``

``
npm test
``

``
npm install --save-dev babel-jest babel-preset-es2015
``

``
npm install babel-preset-env --save-dev
``

**E2E**

``
npm i puppeteer
``

**Configuration**

``
npm install config
``

``
npm install --save-dev cross-env
``
